# HTML Portfolio Website

A simple personal portfolio website built using **only HTML and CSS**
for learning and practicing frontend fundamentals.

## About

This project was created to practice the basics of HTML and CSS by
building a portfolio website from scratch without using any external
libraries, frameworks, or JavaScript.

The website includes:

-   Home/Header section
-   About Me
-   Education
-   Skills
-   Projects
-   Achievements
-   Resume section
-   Contact form
-   Social links
-   Footer

## Technologies Used

-   HTML5
-   CSS3

## Purpose

The primary goal of this project is to strengthen understanding of:

-   Semantic HTML elements
-   Tables
-   Lists
-   Forms
-   Images and hyperlinks
-   Page navigation using anchor links
-   Basic CSS styling
-   Layout, spacing, colors, and typography
-   Responsive design using media queries

## Note

This project is intended **only for practice and learning purposes**. No
CSS frameworks, JavaScript, or backend technologies have been used.
